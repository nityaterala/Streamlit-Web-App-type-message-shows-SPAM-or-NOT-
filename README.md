# 📧 Spam Detection Web App (Streamlit)

Machine Learning NLP project that detects whether a message is spam or not.

## 🚀 Features
- Real SMS Spam Dataset
- TF-IDF Vectorization
- Naive Bayes Model
- Streamlit Web Interface

## ▶️ Run Locally
pip install -r requirements.txt
python model.py
streamlit run app.py
